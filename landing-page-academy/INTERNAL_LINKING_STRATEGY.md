# Internal Linking Strategy for SEO

## Current Internal Link Structure

### Homepage (`/`)

**Links to:**

- `/about` - About page
- `/programs` - Programs page
- `/apply` - Application page (CTA)

### About Page (`/about`)

**Links to:**

- `/` - Homepage
- `/programs` - Programs page
- Departments section (internal anchor)

### Programs Page (`/programs`)

**Links to:**

- `/` - Homepage
- `/about` - About page
- Individual program pages (future)

## Recommended Internal Linking Improvements

### 1. Add Navigation Links

- Ensure all main pages are linked in header navigation
- Add footer links to important pages
- Create breadcrumb navigation

### 2. Content-Based Links

- Link from program descriptions to related content
- Link from testimonials to program pages
- Link from announcements to relevant pages

### 3. Topic Clusters

Create content clusters around main topics:

**Main Topic**: Social Science Education

- **Pillar Page**: `/programs`
- **Supporting Pages**:
  - `/programs/psychology`
  - `/programs/sociology`
  - `/programs/political-science`
  - `/programs/anthropology`
  - `/programs/economics`

### 4. Anchor Text Optimization

- Use descriptive anchor text
- Include keywords naturally
- Avoid generic "click here" links

### 5. Link Depth

- Keep important pages within 3 clicks from homepage
- Use flat site structure where possible

## Implementation Priority

1. **High Priority**: Add footer navigation links
2. **Medium Priority**: Create program detail pages
3. **Low Priority**: Add related content suggestions
